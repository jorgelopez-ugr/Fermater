## Milestone 2:

**Corresponde a la [HU001](./HU001.md)**

### Producto mínimo:
El objetivo de este milestone es ofrecer una verión inicial que permite al usuario obtener un valor resultado llamando **"porcentaje de ganancias"** en función de los parámetros ingrasados por el usuario **"gastado"** y **"kilos recolectados de cada variedad"**.

Un producto simple, sólido e intuitivo:
se le **proporciona** una cantidad recolectada en kg de cada una de las 3 variedades principales.
se le **proporciona** un valor que represente los costes afrontados.
se le **devuelve** un `%` de ganancia o pérdidas en función del precio de HOY.

### Se podrá considerar viable cuando:
1. Exista una forma facil e intuitiva de ingresar los datos por parte del usuario
2. El programa devuelva el porcentaje calculado en la lógica de negocio de este milestone y sea comprobada la veracidad del resultado.
