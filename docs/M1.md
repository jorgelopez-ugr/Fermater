**Primer milestone:**

El objetivo de este milestone es ofrecer una verión inicial que permite al usuario obtener un valor concreto de ganancias en función de los parámetros "gasto" y "kilos recolectados de cada variedad". No requiere de persistencia de datos, no requiere de gestión de ususarios, no requiere de interfaz.

producto sólido e intuitivo:
se le **proporciona** una cantidad recolectada en kg de cada una de las 3 variedades principales.
se le **proporciona** un valor que represente los costes afrontados.
se le **devuelve** un `%` de ganancia o pérdidas en función del precio de HOY.

(mvp básico y robusto, para la primera versión solo queremos el cáclulo que implica captar el precio de hoy, multiplicarlo por lo recogido y compararlo con el coste para obtener un %)
