## Milestone 0:

Este milestone se centra en definir una base técnica para el proyecto.
1. Deben definirse las estructuras de datos que se usarán para almacenar la información.
2. Debe definir de que forma se van a poblar dichas estructuras.
3. Debe sentar las bases para las buenas prácticas de colaboración al repositorio.
4. Debe aportar información sobre la organización del código y las buenas prácticas que se deben tener en cuenta.

Se podrá considerar completado cuando:
- [ ] Las estructuras de datos estén definidas y propiamente documentadas.
- [ ] La recolección e indexado de información debe estar definida y documentada.
- [ ] Se elabore un manuscrito que defina las bases para una colaboración cooperativa y ordenada al repositorio.
- [ ] La organización del repositorio y las buenas prácticas que van a caracterizar el código estén recopilados por escrito en un archivo.
