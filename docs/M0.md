## Milestone 0:

**Corresponde a la [HU001](./HU001.md) [HU002](./HU002.md)**

### Producto mínimo:
Este milestone se centra en definir una base técnica para el proyecto.
1. Deben definirse las estructuras de datos que se usarán para almacenar la información.
2. Debe definir de que forma se van a poblar dichas estructuras.

### Se podrá considerar viable cuando:
1. Las estructuras de datos estén definidas y propiamente documentadas.
2. La recolección e indexado de información debe estar definida y documentada.
