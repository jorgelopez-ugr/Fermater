# Directorio "src"
Aquí debe residir el código fuente de la aplicación, donde se incluirá todo el código aplicado a la lógica de negocio y sus componentes.

## Main del código
El archivo [index.js](index.js) es el punto de entrada o main del proyecto.

## Subdirectorios
- [config](config/): Archivos de configuración que contienen variables, constantes...
